# Generative Chatbot
The EVNTO Chatbot is a conversational AI designed to assist Mansoura University students in engaging with volunteer teams, attending events, and connecting with mentors. The chatbot provides an interactive interface where users can ask questions about upcoming events, register for activities, and get personalized event recommendations.

## Features:
* Answer common questions about events and volunteering opportunities.
* Help users navigate the app and access event details.
* Offer personalized event suggestions based on user preferences.

## Technical Details:
* The chatbot will use huggingface models with a goal to be capable of processing both English and Egyptian Arabic inputs.
* It uses pre-trained models fine-tuned on university-related topics to improve accuracy and relevance in responses.
* The chatbot is integrated with the recommendation system to suggest events based on past interactions.
